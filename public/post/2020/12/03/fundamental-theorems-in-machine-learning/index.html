“Fundamental” theorems in machine learning

Yutong Wang

2020/12/03

There is a nice article on fundamental theorems in mathematics that I really enjoyed reading. Sadly, the section on artificial intelligence only talked about a joke “theorem” from The Big Lebowski. This inspired me to list a few theorems that I personally find to be very pleasing and unifying. The membership criterion for this list is that seeing the theorem should induce the same feeling as seeing this sight:

Vapnik–Chervonenkis theory

  • Sauer-Shelah lemma. Here is a beautiful proof using distributive lattice theory creditted to Peter Frankl and Janos Pach.

Kernel methods

  • Moore–Aronszajn theorem

Low rank approximation

  • Eckart–Young–Mirsky theorem

Spectral clustering

  • Davis-Kahan theorem

Saddle point problems

  • von Neumann’s Minimax theorem

  • Sion’s minimax theorem
